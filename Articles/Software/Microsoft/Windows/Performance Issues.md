2. Change the preferred plan to "High Performance"

## Apply Storage Management suggestions
1. In the Windows search bar, type "Storage settings"
2. Click "Cleanup recommendations"
3. Check the box next to the types of files you'd like to delete, then click "Clean up"

## Close unused browser tabs
- Having many web browser tabs open at once can be very resource-intensive. Close browser tabs you're not using.

## Identify resource-demanding processes in Task Manager
1. Right click on the Windows tool bar, OR search for "Task Manager" in the Windows search bar
2. Click on the "Memory" column header to by how much of your device's memory each process is using. 
3. Close memory-intensive applications that you're not presently using.
